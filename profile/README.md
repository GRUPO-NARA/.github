# NARA — Núcleo de Análise e Recursos Analíticos

> Sempre em busca de informações...

O NARA é um grupo formado por discentes do curso de **Engenharia da Computação** da **Universidade Estadual do Maranhão (UEMA)**, atuando como uma das vertentes do **Laboratório de Engenharia Aplicada (LEA)**. Nosso foco é o estudo, análise e visualização de dados com aplicações reais para o setor público e privado.

## 📂 Projetos

### 🗺️ Mapa Social do Maranhão
Projeto realizado em parceria com o **Ministério Público do Maranhão**, com o objetivo de mapear e visualizar indicadores sociais do estado.

### 📊 Análise de Analfabetismo nos Municípios do Maranhão
Análise de dados do **IPEADATA** sobre os índices de analfabetismo nos municípios maranhenses, gerando insights para políticas públicas.

---

## 🎯 Objetivo do grupo

Promover a análise de dados para o estado do Maranhão, proporcionando ferramentas de análise e desenvolvimento de aplicações **escaláveis**, **confiáveis** e de **fácil manutenção** para o setor público e privado.
